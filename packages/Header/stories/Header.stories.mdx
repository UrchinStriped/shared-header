## Props

| Name                | Required | Type               | Default value | Description |
|---------------------|----------|--------------------|---------------|-------------|
| **links**           | yes      | `IMainNavLink[]`   |  []           | Provides list of top menu items. Wrapping in `NavLink`  |
| **rightSide**       | no       | `ReactNode`        |  undefined    | Render Node on right side of Header |
| **logoDoubleClick** | no       | function           |  () => {}     | Callback for Logo double click |
